# AIS Papers — Self-Supervised / Foundation Models on Vessel Position Data

Curated papers on self-supervised representation learning and dark-ship
detection from AIS trajectory data, plus notes on how each method labels its data.

- `LABELING-METHODS.md` — how each method derives its training signal (auto vs
  third-party), with detailed labeling setups extracted from the Simula
  AIS-shutdown and xView3-SAR papers.
- `MANUAL-DOWNLOADS.md` — URLs/DOIs for papers not on arXiv (journal- or
  GitHub-only).

PDFs are arXiv preprints, included here for personal reference.
